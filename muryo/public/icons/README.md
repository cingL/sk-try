# PWA Icons

This directory should contain PWA icons for the application.

Required icon sizes:
- 72x72.png
- 96x96.png
- 128x128.png
- 144x144.png
- 152x152.png
- 192x192.png
- 384x384.png
- 512x512.png

You can generate these icons from a single source image (recommended: 512x512) using tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Place the generated icons in this directory.
